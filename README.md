# Hi-Tech RH — Site Vitrine

Ce projet utilise **Astro** + **Tailwind CSS** pour créer un site vitrine professionnel.

## Pages incluses
- Accueil
- Nos services
- Contact
- Accès client (SharePoint)

## Démarrer en local

```bash
npm install
npm run dev
```

## Déploiement
Prêt pour être déployé sur **Netlify**.
